# PakUtility

Smart Online Utility Calculators for Pakistan.

## Features
- Electricity Bill Estimator
- Salary Tax Calculator (FBR 2024-25)
- Zakat & Islamic Calculators
- Age, BMI, and daily life tools
- 130+ Utility tools

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
